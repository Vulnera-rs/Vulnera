# Business Model

> **Version:** 1.0 | **Last Updated:** 2025

## Executive Summary

Vulnera is a high-performance, unified security analysis platform that provides comprehensive vulnerability scanning across multiple ecosystems. Built in Rust for superior performance, Vulnera combines dependency analysis, SAST, secrets detection, and API security auditing into a single, developer-friendly platform.

**Key Highlights:**

- **Product:** Multi-ecosystem vulnerability analysis API (v0.3.2)
- **License:** AGPL v3.0 (Open source)
- **Architecture:** Cloud-native, Azure-optimized, container-ready
- **Team:** 4-person core team

## Value Proposition

### Core Value Propositions

1. **Faster Vulnerability Scanning** — Rust performance delivers 50-80% faster analysis than competitors
2. **Unified Security Platform** — One tool replaces multiple point solutions
3. **Superior API Integration** — API-first design with comprehensive OpenAPI documentation
4. **Multi-Ecosystem Support** — 8+ package ecosystems in a single platform
5. **Cost-Effective Solution** — More affordable than enterprise solutions
6. **Memory-Safe Architecture** — Built in Rust for security guarantees
7. **Open Source** — AGPL v3.0 license, community-driven
8. **Privacy-First** — Data encrypted at rest and in transit

### Customer Pain Points Addressed

| Pain Point | Vulnera Solution |
|------------|------------------|
| Slow analysis times | Real-time feedback via Rust performance |
| Tool fragmentation | Unified platform for all security needs |
| Poor CI/CD integration | API-first design with comprehensive docs |
| Limited ecosystem coverage | 8+ ecosystems supported |
| High costs | Affordable pricing for startups and SMBs |
| High false positive rates | Context-aware analysis |
| Complex setup | Docker-ready, configuration via environment variables |

## Target Market

### Primary Segments

1. **Developers/DevOps Teams** — CI/CD integration, pre-commit hooks
2. **Security Teams** — Audits, compliance, risk assessment
3. **Startups** — Affordable, quick setup, compliance preparation
4. **SMBs** — Multi-project management, cost optimization
5. **Mid-Market** — Enterprise features, advanced reporting

### Beachhead Market

**Primary Focus:** Developers/DevOps Teams + Startups

- Product-Led Growth (PLG) aligns with developer-focused approach
- Fast feedback loops for product iteration
- Lower sales friction (self-service signup)

## Pricing Strategy

### Pricing Tiers

| Tier | Price | Scans/Month | Key Features |
|------|-------|-------------|--------------|
| **Free** | $0 | 5 | Public repos, basic dependency analysis |
| **Starter** | $10/month | 500 | All modules, private repos, email support |
| **Professional** | $99-199/month | 2,000 | Custom rules, priority support, API access |
| **Enterprise** | Custom | Unlimited | SSO, on-premise, dedicated support |

### Competitive Position

| Competitor | Pricing | Vulnera Advantage |
|------------|---------|-------------------|
| Dependabot | Free (GitHub only) | Multi-platform, more ecosystems |
| GitHub Advanced Security | $21/user/month | More affordable, not GitHub-locked |
| Snyk | $52-99/user/month | Better performance, lower cost |
| Checkmarx | $10,000+/year | Far more accessible |
